Код товара: 7005

Размеры шкафа:

Высота – 1000 мм.

Ширина – 400 мм.

Глубина – 300 мм.


Шкаф металлический 1-секционный. В стандартную комплектацию каждой секции входят 1 съемная полка, 1 перекладина для вешалок и 5 пластиковых крючков для одежды. Шкаф оснащен встроенным замком и ригельным запиранием. Высококачественное порошковое покрытие.

Возможность окраски в любые цвета по требованию клиента.

Металлические шкафы для одежды могут быть:

Односекционный металлический шкаф состоит из одной секции. В отделении: полка для головного убора, перекладина для вешалки и крючки для одежды, врезной замок.

Двухсекционный металлический шкаф состоит из двух секции. В комплектацию каждой секции входят съемная полка для головного убора, перекладина для вешалки и крючки для одежды.

Трехсекционный металлический шкаф состоит из трех секции. В каждом отделении: полка для головного убора, перекладина для вешалки и крючки для одежды.

Модульный металлический шкаф состоит из неограниченных количество секции по желанию клиента.


Односекционный: Двухсекционный:Трехсекционный:Модульный:

Односекционный с двумя отделениями металлический шкаф состоит из одной секции с двумя отделениями. Шкаф снабжен полкой под головной убор, в отделении перекладина для вешалки и крючки для одежды.

Два деления металлический шкаф состоит из одной секции с двумя отделениями. В каждом отделении: перекладина для вешалки и крючки для одежды.

Четыре деления металлический шкаф состоит из двух секции с 4-мя отделениями. В комплектацию каждого отделения входят перекладина для вешалки крючки для одежды, врезной замок.

Универсальный металлический шкаф состоит из двух секции. В одном отделении: полка для головного убора, перекладина для вешалки и крючки для одежды. В другом: четыре регулируемые по высоте полки.

Двери металлического шкафа могут быть оснащены:

Механический замок EURO-LOCKS (Германия) повышенной секретности на 10 000 комбинаций.

Электронный замок открывающийся силиконовым браслетом, картой. Чтобы открыть такой механизм, нужно просто коснуться браслетом, брелоком или карточкой.

Кодовый замок открывающееся без помощи ключа, при помощи специального кода или пароля. Замок универсален и прост как в монтаже, так и в использовании.

Перекладина и пластиковые крючки:

Каждая секция металлического шкафа имеет перекладину и пластиковые крючки для одежды. Эти комплектации являются необходимыми и достаточными для комфортного хранения сменной одежды в производственных, спортивных и других помещениях.

Дополнительная полка:

В каждой секции имеется полка для головных уборов. В шкафу предусмотрены пошаговые отверстия для крепления полок - 9 см.

Вентиляция:
Шкаф металлический для одежды имеет отверстия для вентиляции. Вентиляционные отверстия предназначены чтобы, одежда вентилировалась и неприятный запах пота и грязной одежды, не застаивался и впитывался, а удалялся через местную систему вентиляции.
Преимущества металлического шкафа для школьника:
• Предназначены для хранения сменной одежды в производственных, спортивных и других помещениях;
• Уникальное сочетание привлекательного дизайна и высоких эксплуатационных характеристик;
• Оборудованы полками, перекладиной для плечиков, крючками для одежды;
• Ключевой замок EURO-LOCKS (Германия) повышенной секретности на 10 000 комбинаций;
• Окраска дверей в другие цвета из палитры;
• Специальная обработка шкафов перед покраской повышает коррозийную стойкость шкафов. При покраске локеров используется гладкое порошковое покрытие, стойкое к появлению царапин;
• Цельнометаллическая конструкция;
• Высококачественная сталь толщиной 1 мм;
