Держатель рамы Клипса

Код товара: 4060

Держатель для рамы изготавливается сам держатель из прозрачного пластика, а прищепка для крепления из пластика белого цвета.

Держатель нужен для выставления информации в раме.

Держатель не заменим, когда нужно раму закрепить к трубам, полкам, корзинам, разным решеткам и т.д.

Держатель рамы Полочный магнитный

Код товара: 4061

Материал изготовления самого держателя прозрачный пластик, основание магнитное из белого пластика. Магнитный полочный держатель рамы нужен для крепления рамы к металлической поверхности, например полки из металла на стеллажах.

В комплектацию держателя входит крепления для рамы, из стойки небольшого размера изогнутой формы, и магнитный фиксатор, через который держатель и закрепляется к металлической поверхности.

Данный держатель дает возможность установки рамы и на горизонтальную, и на вертикальную металлическую поверхность, так как крепление для рамы подвижное, раму можно поворачивать и устанавливать в нужном ракурсе.

Держатель рамы Петля

Код товара: 4063

Материал изготовление держателя – пластик белого цвета. Данный держатель предназначен для подвешивания рамы. В комплектацию данного держателя входит крепления для рамы и петля, через который и держатель подвешивается.

Данный держатель позволяет разместить рамки с информацией на любые крючки.

Для более удобного подвешивания данный держатель можно подвешивать на тонкий трос.

Держатель рамы Корзина

Код товара: 4065

Материал изготовления самого держателя рамы – пластик, а материал стойки держателя – алюминий, на стойке есть крепление, который изготовлен из пластика серого цвета.

Для того чтобы плотно закрепить держатель на решетчатым оборудований на держателе есть пружина, которую зажав и разжав можно просунут крепления держателя между прутками.

Данный держатель предназначен для крепления к корзинам и решеткам, можно регулировать расстояние между креплениями благодаря пружины от 7,5 см до 23 см.
