4-гранные торговые витрины и прилавки



  • Торговый прилавок без порога

    Код товара: 5001

    Торговый прилавок без порога состоят из: профильного каркаса из алюминия, заполнение идет стекло.

    Так как прилавок полностью идет прозрачный, дает самое максимальное место для демонстрации товара.

    Может служит как и прилавок, так и как рабочее место продавца.

  • Торговый Прилавок с порогом

    Код товара: 5002

    Торговые прилавки с порогом состоят: из профильного каркаса из алюминия, верхнее заполнение идет стекло, нижнее ДВП + пленка. Цвет пленки можно подобрать.

    Порог снизу нужен в основном как для временного хранения товара и разных других вещей от взгляда покупателей. Высоту нижней закрытой части можно изготавливать по разной высоте, но стандартная высота идет 250 мм, 400 мм, 650 мм.

    Код товара: 5003

    Торговые витрины без порога состоят из: профильного каркаса из алюминия, заполнение идет стекло.

    Код товара: 5004

  • Торговая витрина с экспопанелью

    Код товара: 5005

    Торговые витрины с экономпанелью: из профильного каркаса из алюминия, задняя стенка из экономпанели, верхнее заполнение идет стекло, нижнее ДВП + пленка. Цвет пленки можно подобрать.

    Комплектацию витрины можно дополнить дверями, подсветкой, замком и дополнительными полками. Для рекламы можно доукомплектовать верхним, средним, нижним порогом.
