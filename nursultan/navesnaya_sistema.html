Готовые комплекты:

Конструкция навесной системы хранения для офисов, домов:

• Несущий рельс - основа для крепления стоек, которые, в свою очередь, являются базой для размещения полок и корзин. Несущие рельсы выполняются из листовой стали и окрашиваются порошковой краской. Несущий рельс крепится непосредственно к стене, на него навешиваются стойки.
• Стойки для монтажа кронштейнов располагаются на несущем рельсе и при необходимости могут дополнительно крепиться к стене. На стойке имеется два ряда пазов для крепления специальных кронштейнов под полки и перекладины.
• Переходник для стойки изготовлен из стали, обладает высоким запасом прочности. Полимерно-порошковое покрытие устойчиво к механическому воздействию и истиранию, защищает изделие от ржавчины.

Полки, корзины:

• Полка сеточная металлическая - один из основных компонентов системы для размещения одежды и прочих предметов гардероба.
• Выдвижная корзина - практичный и удобный аксессуар, который не только обеспечивает свободный доступ к содержимому, но и позволяет сберечь свежесть вещей при полной загруженности шкафа.
• Полка металлическая - один из основных компонентов для размещения одежды и прочих предметов гардероба.
• Полка для обуви. Благодаря небольшому уклону и специальному зонированию предметы компактно располагаются и занимают меньше пространства.
• Выдвижная рама для мелкосетчатых корзин предназначена для установки мелкосетчатой корзины.
• Мелкосетчатая корзина для стеллажной и гардеробной системы Титан-GS изготовлена из выосокачественной стали.
• Стационарная корзина. Точно продуманные размеры конструкции позволяют с легкостью находить вещи и максимально эффективно использовать пространство.
• Полка универсальная подходит для хранения небольших предметов. Боковая конструкция создает дополнительное функциональное пространство. Изготовлена из стали, обладает высоким запасом прочности.

Комплектация металлической навесной системы:

• Кронштейн для установки полок/корзин устанавливается в перфорацию стойки на удобной высоте с возможным шагом крепления 32 мм.
• Металлическая перекладина для вешалок - один из основных компонентов системы для размещения одежды и прочих предметов гардероба. Размещается горизонтально на подвесные кронштейны на нужном расстоянии.
• Кронштейн для установки перекладины навешивается на кронштейн для полки и представляет собой крючок, который выполняется из листовой стали с порошковым покрытием.
• Разделитель / Боковина корзины Дополнительное функциональное решение, которое сделает вашу корзину для вещей еще более вместительной и практичной.
• Крючок Металлический одинарный крючок для перфорированной панели.
• Крючок боковой подходит для хранения небольших предметов. Боковая конструкция создает дополнительное функциональное пространство.
• Пластиковый соединитель для перекладины используется для наращивания длины перекладины (штанги) гардеробной системы.
• Торцевые заглушки для перекладины используется для придания максимального эстетического вида готовой гардеробной системе.
